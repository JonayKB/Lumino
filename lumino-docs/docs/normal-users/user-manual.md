# Manual para el Usuario

!!! info "Bienvenido"
    Bienvenido a **Lumino**, la plataforma educativa donde puedes matricularte en módulos, seguir lecciones y consultar tus calificaciones.

Lumino permite a estudiantes y profesorado gestionar módulos, lecciones y evaluaciones. Este documento te guía paso a paso por todas las funcionalidades disponibles.

---

# Acceso y registro

## Iniciar sesión

1. Accede a la URL principal de la plataforma.
2. Selecciona **Login**.
3. Introduce tu *nombre de usuario* y *contraseña*.
4. Pulsa **Sign in**.


!!! warning
    Si introduces credenciales incorrectas recibirás el mensaje:  
    **Incorrect username or password.**(1)
    { .annotate }
    
       1. En español recibiras: **Nombre de usuario o contraseña incorrecta**

## Crear una cuenta (solo alumnado)

Véase:

!!! danger "Campos obligatorios"
    - Nombre de usuario  
    - Contraseña  
    - Nombre  
    - Apellidos  
    - Correo electrónico  

Tras el registro verás el mensaje:

> Welcome to Lumino. Nice to see you! (1)
> { .annotate }
> 
1. En español veras: **Bienvenido a Lumino. Encantado de conocerte**

# Módulos (Subjects)

## Ver mis módulos

Al acceder a **My subjects**, verás un listado con todos los módulos matriculados.


## Solicitar certificado

!!! note "Requisitos"
    Puedes solicitar tu certificado únicamente cuando **todos tus módulos tienen nota asignada**.


Al solicitarlo:

* Se generará un PDF con tus calificaciones.
* Recibirás un correo electrónico con el documento adjunto.
* Verás el mensaje:
  **You will get the grade certificate quite soon at [tu_correo@example.com](mailto:tu_correo@example.com)**(1)
    { .annotate }
    1. En español recibiras: **Obtendras tu certificado de notas pronto en [tu_correo@example.com](mailto:tu_correo@example.com)**

---

# Lecciones (Lessons)

Dentro de un módulo puedes consultar todas las lecciones disponibles. Cada una contiene:

* Título
* Contenido en markdown
* Fecha de creación (si aplica)

!!! tip
    Usa el navegador interno del módulo para moverte entre lecciones de forma rápida.
